    This is a practice portfolio project.

    -Consumes spells' data from: "https://www.dnd5eapi.co/api/spells". Then passes to UI.

    -For a spell description: "https://www.dnd5eapi.co/api/spells/[spell-index]" E.G:"https://www.dnd5eapi.co/api/spells/aid"

    -Fully responsive page.

    -Spells inserted to a Trie data structure, for efficient searching operations.

    -Search bar with auto-completion feature.

    -Used API is free: "https://www.dnd5eapi.co/"

    -API contains just PHB spells.
